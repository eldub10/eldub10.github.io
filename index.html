DiCE

2010

The Flashlight Digital Cable Emulator (DiCE) is a Linux-based video streamer that enables software developers to build and test interactive TV applications. DiCE delivers MPEG-2 transport streams (both broadcast and on-demand formats) as well as interactive applications and program specific information (PSI) to cable devices via QAM. DiCE provides a web interface to configure streams and applications.

EBIF User Agent

2008

Enhanced Binary Interchange Format (EBIF) is a specification for software applications that provide interactivity on most legacy set-top boxes. EBIF applications can be delivered over an RF stream to any set-top box with an EBIF user agent (similar to a web browser). The user agent renders the EBIF markup into graphics on the end user’s television. The EBIF standard also enables two-way communication between the set-top box and backoffice servers. Flashlight Engineering developed and deployed an EBIF user agent on Motorola set-top boxes for Cox Communications.

Vidiom Systems VISION Workbench was an Open Cable Application Platform (OCAP) Java development environment based in Eclipse. Jam Player was based on Supertracks’ audio streaming technology that delivered music from cached, secure files. This method was a cost-effective way to deliver music over the Internet while providing CD-quality music playback. It also enabled new advertising and promotion models incorporating demographic targeting and detailed tracking. The Supertracks platform was a modular design utilizing a framework architecture based on COM. Component DLLs were created in C++ and user interface components were created in HTML/Javascript. This was deployed by Microsoft, Virgin and EMI in a public beta with Portland radio station Jammin 95.5, and hip-hop website XTuner.
